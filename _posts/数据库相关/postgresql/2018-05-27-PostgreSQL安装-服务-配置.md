# 使用yum库设置安装postgresql94版本，最好先创建用户组和用户 -->

#### POSTGRESQL 开放外网IP访问

```
# 编辑pg_hba.conf ->设置访问权限
# 允许所有IP访问
host  all  all  0.0.0.0/0   md5

# 编辑postgresql.conf ->监听所有IP
listen_addresses = '*'	# what IP address(es) to listen on;
```
