#### [OpenClaw简介]()
OpenClaw（曾用名Clawdbot、Moltbot）是一个本地优先的AI智能体框架，核心价值在于将大语言模型的推理能力与本地操作系统深度绑定，实现从"对话式AI"到"行动式AI"的转变。它就像一位住在您电脑里的全能助理，能通过微信、Telegram、飞书等聊天软件接收指令，自动完成文件整理、邮件处理、浏览器操作等实际任务。

#### 快速本地安装（手动安装）
鉴于网络不稳定，采用收到安装模式，安装前需要搭建node.js开发环境。

```
# node指令下载安装openclaw包
npm install -g openclaw@latest

# 执行配置向导
openclaw onboard --install-daemon


```

#### 常用指令集合

- 配置向导 ：openclaw onboard --install-daemon
- 快速检查：openclaw doctor
- 网关检查：openclaw status  、openclaw health  、 openclaw gateway status 
- 打开仪表板：`openclaw dashboard`
